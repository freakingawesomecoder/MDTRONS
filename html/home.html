For HTML documents (and the text/html MIME type), the following processing model must be followed to determine what the indicated part of the document is. Parse the URL, and let fragid be the component of the URL. If fragid is the empty string, then the indicated part of the document is the top of the document. If there is an element in the DOM that has an ID exactly equal to fragid, then the first such element in tree order is the indicated part of the document; stop the algorithm here. If there is an a element in the DOM that has a name attribute whose value is exactly equal to fragid, then the first such element in tree order is the indicated part of the document; stop the algorithm here. Otherwise, there is no indicated part of the document.
